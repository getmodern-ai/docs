## About Jamf Pro

Jamf Pro is a mobile-device-management (MDM) platform used to deploy, configure and secure macOS, iOS, iPadOS and tvOS devices. Connecting Jamf to Modern lets your IT team trigger remote commands, pull inventory or enforce compliance policies directly from chat.

## What the Jamf integration enables

| Capability           | Description                                            |
| -------------------- | ------------------------------------------------------ |
| Access Management    | Create, update, and manage users and their permissions |
| Automation workflows | Streamline device provisioning and management          |

Anything defined in the [Jamf API](https://developer.jamf.com/jamf-pro/reference/classic-api) can be accessed through Modern.

### 3. Configure the integration in Modern

* In Modern, navigate to Applications and then the Available tab. Find the Jamf integration and press Connect.

<Frame>
  <img src="/images/jamf_serval_modal.png" alt="Modern Jamf connection modal" data-og-width="2466" width="2466" data-og-height="1494" height="1494" data-path="images/integrations/jamf/jamf_serval_modal.png" data-optimize="true" data-opv="3" srcset="/images/jamf_serval_modal.png 280w, /images/jamf_serval_modal.png 560w, /images/jamf_serval_modal.png 840w, /images/jamf_serval_modal.png 1100w, /images/jamf_serval_modal.png 1650w, /images/jamf_serval_modal.png 2500w" />
</Frame>

* The Service Instance ID should be the full domain of your Jamf Cloud instance. For example, if your Jamf Cloud URL is `https://modern.jamfcloud.com`, you should enter `modern.jamfcloud.com` here.
* Copy over the client ID and client secret from the previous section.
* Leave the scopes blank and hit `Save`. That's it! Jamf should now be connected.

***

## Modern Configuration

1. In Modern go to **Apps → Available → Jamf → Connect**.
2. Outlined in step 3.3, enter your **Service Instance ID** (your Jamf Cloud domain, e.g., `modern.jamfcloud.com`), **Client ID** and **Client Secret** from the API Client you generated above.
3. Click **Save**.

Modern will now use the Jamf OAuth token it generates to run your workflows. Rotate the client secret anytime—just update it in Modern by reconnecting the app.
