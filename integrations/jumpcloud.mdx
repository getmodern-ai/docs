## About JumpCloud

JumpCloud is a cloud-based directory platform that enables secure, frictionless access to resources. Connecting JumpCloud to Modern allows you to automate user provisioning, manage device access, and enforce security policies across your organization.

## What the JumpCloud integration enables

| Capability           | Description                                            |
| -------------------- | ------------------------------------------------------ |
| Access Management    | Create, update, and manage users and their permissions |
| Automation workflows | Streamline provisioning and lifecycle management       |

Anything defined in the [JumpCloud API](https://docs.jumpcloud.com/api/index.html) can be accessed through Modern.

***

## JumpCloud Configuration

The Modern JumpCloud integration requires you to create an API key in your JumpCloud Admin Portal. This key allows Modern to authenticate and manage your JumpCloud resources.

### Generate your JumpCloud API key

1. Sign in to the JumpCloud Admin Portal at [https://console.jumpcloud.com](https://console.jumpcloud.com).
2. In the top-right corner, click your avatar (initials) and choose "My API Key".
3. Click "Generate New API Key". A 40-character token will appear.
4. Copy the key immediately and store it securely. The key will not be shown again after you close the dialog.
5. Click "Close" to finish.

## Modern Configuration

Enter the following information in the Modern UI:

| Field         | Description                                                                                                                                                                    |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Instance Name | This is the name of the JumpCloud instance. You can use any value here. It will be displayed in the Modern UI and must be unique within your Modern team for this integration. |
| API Key       | This is the API key generated in the JumpCloud Admin Portal.                                                                                                                   |

Submit the form to complete the integration.
