## About PandaDoc

PandaDoc is a document automation platform that streamlines the creation, delivery, and e-signature process for contracts, proposals, and other business documents. Connecting PandaDoc to Modern enables automated document workflows, contract management, and signature collection directly from your service desk.

## What the PandaDoc integration enables

| Capability           | Description                                                                           |
| -------------------- | ------------------------------------------------------------------------------------- |
| Access Management    | Create, update, and manage users and their permissions within your PandaDoc workspace |
| Automation workflows | Streamline document creation, sending, tracking, and signature collection processes   |

Anything defined in the [PandaDoc API](https://developers.pandadoc.com/reference/about) can be accessed through Modern.

***

## PandaDoc Configuration

### Prerequisites

* You must have admin permissions in your PandaDoc workspace to access API settings
* Access to PandaDoc's API and Integrations section

### Create an API Key

The following steps are defined in the [PandaDoc API Reference](https://developers.pandadoc.com/reference/about) with requirements defined [here](https://developers.pandadoc.com/reference/api-key-authentication-process).

1. In PandaDoc, click the **gear icon** (Settings) in the bottom left corner of the screen
2. Under your current workspace section, click **API and Integrations**
3. Scroll down to the **API and Webhooks** section
4. Click the **API** tile, then click **Open**
5. Copy the **API Key** and save it securely for later use

***

## Modern Configuration

1. In Modern go to **Apps → Available → PandaDoc → Connect**

2. Enter the following information:

   | Field            | Description                                                                 |
   | ---------------- | --------------------------------------------------------------------------- |
   | Integration Name | A friendly name for this integration (e.g., "PandaDoc Document Management") |
   | API Key          | The API key from your PandaDoc workspace                                    |

3. Click **Save**
