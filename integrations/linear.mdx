## About Linear

Linear is a modern project and issue tracking tool designed for software development teams. It offers streamlined workflows, fast performance, and integrations that help engineering teams plan, track, and ship products efficiently.

## What the Linear integration enables

| Capability           | Description                                                                                                                                            |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Automation workflows | Create, update, archive, and manage issues. Add comments and labels, update priorities and projects, and coordinate work across your development team. |
| Ticket Syncing       | Sync tickets between Modern and Linear                                                                                                                 |

Anything defined in the [Linear API](https://developers.linear.app/docs) can be accessed through Modern.

***

## Linear Configuration

Connecting Linear to Modern is straightforward and requires no manual configuration on the Linear side. The integration uses OAuth to securely connect your Linear workspace with Modern.

### Prerequisites

* You must have admin permissions in your Linear workspace
* Access to authorize applications in your Linear organization settings

***

## Modern Configuration

1. In Modern go to **Apps → Available → Linear → Connect**
2. Click the **Connect** button to initiate the OAuth flow
3. A new window will open directing you to Linear's authorization page
4. Review and accept the integration permissions
5. Click **Authorize** to complete the connection
6. You'll be redirected back to Modern with the integration successfully connected

That's it! No additional configuration is needed. Modern can now access your Linear workspace through the authorized OAuth connection.
