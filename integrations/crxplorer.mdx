> Analyze and scan Chrome browser extensions for security risks

## About CRXplorer

CRXplorer is a browser extension security platform that enables organizations to analyze Chrome extensions for potential security risks, malware, and privacy concerns. Connecting CRXplorer to Modern enables automated extension scanning workflows and security monitoring for your organization's browser environment.

## What the CRXplorer integration enables

| Capability           | Description                                                      |
| -------------------- | ---------------------------------------------------------------- |
| Extension scanning   | Analyze Chrome extensions for security vulnerabilities and risks |
| Automation workflows | Automate extension approval processes and security reviews       |

***

## CRXplorer Configuration

### Prerequisites

* You must have an active CRXplorer account

### Create an API Token

1. Log in to your [CRXplorer account](https://crxplorer.com/account)
2. Click **Create New Token**
3. Copy the generated API token

<Warning>
  Store your API token securely. You won't be able to view it again after closing the dialog.
</Warning>

***

## Modern Configuration

1. In Modern, go to **Apps → Available → CRXplorer → Connect**

2. Enter the following information:

   | Field     | Description                               |
   | --------- | ----------------------------------------- |
   | API Token | The API token from your CRXplorer account |

3. Click **Submit**

<Check>
  Integration configured! Modern will verify your credentials and establish the connection.
</Check>
