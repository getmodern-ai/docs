## About incident.io

incident.io is an incident management platform for tracking and responding to incidents.

## What the incident.io integration enables

| Capability           | Description                                                                        |
| -------------------- | ---------------------------------------------------------------------------------- |
| Automation workflows | Create incidents, update status, and query incident data through incident.io's API |

Access any endpoint from the [incident.io API](https://api-docs.incident.io/) through Modern workflows.

## incident.io Configuration

### Prerequisites

* You must have admin permissions in your incident.io organization to access API settings
* Access to incident.io's API and Integrations section

### Create an API Key

1. In incident.io, click the **gear icon** (Settings) in the bottom left corner of the screen
2. Under your current organization section, click **API and Integrations**
3. Scroll down to the **API and Webhooks** section
4. Click the **API** tile, then click **Open**
5. Copy the **API Key** and save it securely for later use

## Modern Configuration

1. In Modern, navigate to **Apps → Available → incident.io → Connect**
2. Enter an optional **Instance Name** for this integration (e.g., "Production incident.io")
3. Enter the **API Key** you created in incident.io
4. Click **Save**
