This section contains documentation for all available Modern integrations.

Modern integrates with identity providers (Okta, Google Workspace), collaboration tools (Slack, GitHub), IT service management (Jira, Freshservice), device management (Jamf, Kandji), cloud infrastructure (AWS), and other business tools. You can also connect custom applications through our API integration framework.

Each integration guide includes setup instructions, available actions, and configuration examples.

## Duplicate Applications Across Teams

You can copy a connected application's configuration to other teams in one click. This is useful when multiple teams need the same integration setup — instead of configuring each team from scratch, duplicate the application and adjust as needed.

To duplicate: Navigate to the application in your team's **Applications** settings, click the menu (**...**), and select **Duplicate to Team**. Choose the destination team and the application configuration is copied over.

## Integration Sync History

Monitor the health of your connected integrations with detailed sync history logs. Each application's settings page includes a sync history view where you can:

* **Track recent sync runs** — See when each sync ran and its outcome
* **View breakdowns** — Check how many items were added, updated, or removed in each sync
* **Identify issues** — Spot failed syncs and review error details to troubleshoot connectivity or permission problems

Navigate to **Applications** → select an application → **Sync History** to review.
