Use `snippets/modern-style-guide.css` as the only source of truth.

## Scope

This setup is light-theme only.
Dark mode tokens and `.dark` overrides are intentionally removed.

## Maintain these parts

- `@theme` token blocks
- Semantic color mappings under `/* Primary Colors - Light Theme */`
- Utility classes (`.custom-scrollbar*`, `.bg-glass`, `.loading-*`, `.auraAnimate`, `.shine-text`)
- Editor/terminal selectors (`.cm-*`, `.xterm-*`)

## Copy command

```bash
cp snippets/modern-style-guide.css path/to/your-app/src/styles/globals.css
```
